# Holorb

A fast, one-thumb hyper-casual game built in **Godot 4**. Roll your orb and slip
through the hole in each oncoming wall — survive as long as you can. Collect
coins during runs and spend them on procedurally-generated ball skins.

- Portrait, touch-first (drag to move)
- Endless, speed ramps with score
- Coin pickups + skin shop
- Rewarded-ad coins, gated interstitials, remove-ads IAP (see `ANDROID_SETUP.md`)

Built for Android.
